# DOMINO Chain

Red social gamificada de retos diarios en video de 15 segundos donde los usuarios crean reacciones en cadena globales visibles en un mapa 3D interactivo. Dirigida a la Generación Z (15-25 años) con mecánicas de nominación viral y ranking de impacto.

Generado y mantenido automáticamente por **[Maris AI](https://www.marisai.es)**. Cada vez que actualizas la app desde Maris AI, este repo recibe un commit nuevo en `main` con la versión actual del frontend.

## Estructura

- `frontend/` — React + Vite + Tailwind

## Cómo correrlo en local

```bash
cd frontend
npm install
npm run dev
```
